# ELION Lab

**Elucidating Language Intelligence & Representation**

ELION Lab is a research group at Konkuk University, led by Prof. Jaehyung Seo. We explore the frontiers of natural language processing and AI systems, aiming to understand how language models represent knowledge, reason, and generate meaning toward interpretable, controllable, and trustworthy language intelligence.
